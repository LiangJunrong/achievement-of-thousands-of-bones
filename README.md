## 实现功能

* [x] 新增任务
  * [x] 输入任务 + 点击新增按钮
  * [x] 输入任务 + 键盘点击回车

* [x] 修改任务 - 点击选项进行修改，点击其他地方完成修改

* [x] 回收任务 - 点击未完成或者已完成的循环按钮，可以将任务放至回收站

* [x] 删除任务 - 点击回收站的删除按钮，彻底删除任务
